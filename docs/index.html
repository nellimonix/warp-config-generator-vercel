Первое использование: Для работы требуется GitHub токен с правами на repository dispatch. Как получить токен?

Как получить GitHub токен:

  1. Перейдите в GitHub Settings → Personal Access Tokens
  2. Нажмите "Generate new token (classic)"
  3. Выберите scope: repo
  4. Скопируйте токен и вставьте ниже
